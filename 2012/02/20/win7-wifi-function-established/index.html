Win7建立wifi功能

想好好利用下手机的wifi功能，手机连接电脑上网吗？

手机wifi是不是搜索不到笔记本建立的临时网络？

有两台电脑都想上网，只有一个宽带，没有路由器，怎么办？

现在就告诉大家一个简单有效的方法，在笔记本上建立一个wifi热点，供wifi手机或者其他笔记本链接

综述：其实就是三段代码:（括号内部分）

1.（netsh wlan set hostednetwork mode=allow ）

2.（netsh wlan set hostednetwork ssid=mywifi key=12345678 ）3.(netsh wlan start hostednetwork)

解释下，很简单，第一个就是设置网络承载模式为允许，第一个设置您建立的wifi用户名及密码，ssid=后面就是用户名，key=后面是密码，自己设定，第三个就开启了。

接下来就是具体操作了：

1.打开WIN7开始菜单，在左下角的框上输入CMD，搜索出来的CMD.EXE对着它右键，选择以“管理员身份运行”。
2.依次输入以下代码，红色部分，每输入完一次点击一次回车键

（1）netsh wlan set hostednetwork mode=allow
承载网络模式设置为允许
（2）netsh wlan set hostednetwork ssid=mywifi key=123456789 设置一个名字为mywifi的无线接入点，接入点密码为123456789（3）netsh wlan start hostednetwork

启动承载网络

如图：

然后点击任务栏的网络图标，如果出现下图的mywifi接入点，恭喜您建立成功


接下来可能还需要一些设置
进入“网络和共享中心”在这个位置把两个网络（一个是自己建立的mywifi无线链接，一个是已经联网的，我的是宽带）都设置为家庭网络

如图

 

此时mywifi访问类型是无法连接到internet，记住mywifi的连接是 无线网络链接2，下面用用到 我们需要下面的操作

点击宽带链接——》属性——》共享

在 家庭网络链接 下拉选项中选择无线网络链接2，这个是根据上面的mywifi所属的链接选择，并且勾选上允许其他连接。。。。（第三个，有些没有中间的一个，没关系，不用管，有的全勾上就可以）然后确定再看看网络共享中心的链接，

如图

大功告成

通过手机或者其他笔记本访问这个mywifi链接试试

当然，是不是觉得每次都要输入代码，很麻烦，没关系，下面是偷懒的办法。前提是您第一次建立成功

新建一个记事本，把第三段代码复制到记事本里，如图

![clip_image013[4]](http://blog.zhaoyue.info/zqxjd-content/uploads/2012/02/clip_image0134.gif)

然后把记事本的后缀txt更改为bat，这样您下次只需要双击下这个文件，就可以开启您的wifi热点了，是不是很简单。

补充： 实践中遇到一些问题是，手机能搜素的到，并且已经链接（或者无法链接），但是手机还是无法上网。不用着急，下面就是解决办法

打开网络共享中心，双击无线网络链接2（就是mywifi的连接），然后点击属性

![clip_image015[4]](http://blog.zhaoyue.info/zqxjd-content/uploads/2012/02/clip_image0154.gif)

再在如上图中下拉双击选择“Internet 协议版本 4（TCP/IPv4）” 如下图

![clip_image017[4]](http://blog.zhaoyue.info/zqxjd-content/uploads/2012/02/clip_image0174.gif)
IP和子网掩码按照上图填写。而DNS就根据您自己已经链接的那个链接的详细信息里面的Ip V4 DNS服务器上的填写。我的是宽带链接，点击宽带链接——》详细信息，照着填写就可以了

如图

![clip_image018[4]](http://blog.zhaoyue.info/zqxjd-content/uploads/2012/02/clip_image0184.jpg)

电脑上的设置就到这里了。接下来是手机设置
在手机待机桌面上按菜单键–设置–无线和网络–WLAN 设置–按菜单键–高级–使用静态IP–IP地址（192.168.2.2)–网关（192.168.2.1）–网络掩码（255.255.255.0）–域名1（你电脑上的第一个DNS）–域名2（你电脑上的第二个DNS）–返回键
![clip_image019[4]](http://blog.zhaoyue.info/zqxjd-content/uploads/2012/02/clip_image0194.jpg)
现在，你可以连接名称为mywifi密码是123456789的WIFI接入点了
