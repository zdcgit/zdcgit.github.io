使用quartus II创建一个工程

环境：quartus II版本: 12.0sp1

1、打开quartus II软件，如图1所示（软件设置默认）。创建新的工程，选择Create a New Project；如果打开已有的项目，选择Open Existing Project；如果打开最近使用的项目，直接选择Open Recent Project下面显示的即可。当然，这些操作也可以在打开界面，选择“File”，下拉选项里面选择。



2、在这里，创建一个新工程，选择Create a New Project，选择之后如图2所示，然后选择Next。

3、第二步之后，会看到图3所显示界面，第一行是选择Project存放的位置，路径中不要有中文和空格；第二行是Project的名字；第三行是顶层模块名，一般就与工程名一样，然后选择Next。

4、在图4界面中，如果添加已有的file，点击File name后面的浏览，选择所需file即可，在点击旁边Add或者Add All，在下面大方框中就会显示已经添加的files，然后选择Next。

5、在图5界面中，选择器件型号，先选择Family中器件系列，如果知道器件的Package,Pin count,Speed grade，则选择旁边的Package,Pin count,Speed grade，然后在下面Available devices中选择器件型号，再选择Next。

6、在图6所示界面中，选择仿真工具和语言，可以略过，选择Next。

7、在图7所示界面中，选择Finish。

8、然后选择工具栏上的new，或者“File”菜单里面的New，如果8所示，这里选择Design Files中的Verilog HDL file，双击或者单击之后点击OK。

9、输入代码之后，Ctrl+S保存，保存名字要与实体名一致，如图9所示，然后点击工具栏上的Analysis & Synthesis（分析与综合），之后报告无错误，则点击工具栏上的Pin Planner（管脚分配），在管脚分配中的Location下面输入对应的管脚编号，分配完之后，再点击工具栏上Start Compilation(编译) ，报告无错误则变成成功。创建工程完成，再就是烧写程序到芯片中了。
