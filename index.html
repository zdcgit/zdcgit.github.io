Win7建立wifi功能

想好好利用下手机的wifi功能，手机连接电脑上网吗？

手机wifi是不是搜索不到笔记本建立的临时网络？

有两台电脑都想上网，只有一个宽带，没有路由器，怎么办？

现在就告诉大家一个简单有效的方法，在笔记本上建立一个wifi热点，供wifi手机或者其他笔记本链接

综述：其实就是三段代码:（括号内部分）

1.（netsh wlan set hostednetwork mode=allow ）

2.（netsh wlan set hostednetwork ssid=mywifi key=12345678 ）3.(netsh wlan start hostednetwork)

解释下，很简单，第一个就是设置网络承载模式为允许，第一个设置您建立的wifi用户名及密码，ssid=后面就是用户名，key=后面是密码，自己设定，第三个就开启了。

鼠标右键添加打开CMD窗口

1.在空白处右键添加

打开注册表，找到HKEY_CLASSES_ROOT\Directory\Background\shell，在shell上右键，新建->项，名字修改为CommandPrompt,修改右侧的默认值(双击默认)，在“数值数据”一项输入“打开CMD”；
再在CommadPrompt下新建子项，名字修改为command，修改右侧的默认值为“C:\windows\system32\cmd.exe”。

##2.在文件夹上右键添加
打开注册表，找到HKEY_CLASSES_ROOT\Folder\shell，然后就和1同样操作。

##3.在文件上右键添加
打开注册表，找到HKEY_CLASSES_ROOT*\shell，然后就和1同样操作。

注：关于“c:\windows\system32\cmd.exe”这个值，有的说是为“cmd.exe cd “%1””，还有其它写法，我自己是按照“c:\windows\system32\cmd.exe”这个设置成功的。

Word中常见问题解决方法

1、分页
将光标放在上一页最后一行，点击页面布局->分隔符->分节符->下一页。

2、插入页码
点击插入->页码->设置页码格式->页码编号选中起始页码。

3、目录页码不能自动对齐
点击引用->目录->自定义目录，然后在“常规”的“格式”中选择“正式”，生成目录之后，
再次进入自定义目录，在“格式”中选择“来自模板”，即可。
